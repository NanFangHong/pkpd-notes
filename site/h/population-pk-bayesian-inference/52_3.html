52.3 Projection language for model building
Model building can be read as changing the approximating family. Adding a
covariate expands the family. Adding inter-occasion variability expands the
family. Changing residual error geometry rotates the family. Adding a mecha-
nistic PD state changes the forward map. Diagnostics tell us where the current
family is failing to approximate the observed distribution.
