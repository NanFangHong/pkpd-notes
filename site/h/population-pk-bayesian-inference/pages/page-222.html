55.6 Model selection
The 6.008 model-selection unit prepares the reader for the PopPK reality that no
model is true. A structural PK model is an approximation. A residual model is
an approximation. A covariate model is an approximation. The question is not
whether a model is perfect, but whether it is useful, calibrated, and interpretable
for the task.
This view reduces anxiety about model comparison. OFV, AIC, BIC, cross-
validation, VPC, and clinical utility are not enemies. They are dierent sum-
maries of how an approximating family behaves.
