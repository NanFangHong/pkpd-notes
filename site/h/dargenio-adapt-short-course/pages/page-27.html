D’Argenio 的 debugging 哲学非常朴素：always simulate rst。不要一上来就跑
individual 或 population estimation。先确认模型在 nominal parameters 下能
正常 simulation。

ADAPT 可以把 intermediate results 写到 screen 或 run le。复杂模型的
ODE 会被求解成百上千次；一个中间步骤出现 not a number，就可能表现成很
难理解的 solver error。因此要主动检查：
– square root 里的表达式是否可能因数值误差变成负数；
– denominator 是否可能为零；
– conditional branch 是否覆盖 T = 0；
– parameter values 是否越界；
– dose/input 是否真的进入了预期 compartment。
