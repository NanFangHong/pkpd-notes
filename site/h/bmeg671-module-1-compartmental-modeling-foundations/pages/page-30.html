6.4 Step-input code pattern
The corresponding live script should be read as a miniature experiment. The
input rate is not hidden in the initial condition; it lives inside the right-hand
side:
t = 0:0.1:10;
k = 1;
IR = 1;
qi = 0;
soln = ode45(@(t,y) ODEs(t,y), t, qi);
q_sol = deval(soln,t,1);
q_ss = IR/k;
with
function derivVector = ODEs(t,y)
k = 1;
IR = 1;
q = y(1);
dqdt = IR - k*q;
derivVector = [dqdt]';
end

later plot may look smooth and still be answering the wrong question.
