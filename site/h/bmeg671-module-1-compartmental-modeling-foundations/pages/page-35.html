7.4 Impulse versus step in code
The transcript’s distinction is worth making explicit:
impulse dose ⇒ put dose in initial condition,
step input ⇒ put rate inside ODE function.
If an impulse is coded as a nite input at every time step, the simulation adds
too much mass. If a continuous infusion is coded only as an initial condition,
the input stops immediately. These are not small coding choices; they change
the experiment being simulated.
