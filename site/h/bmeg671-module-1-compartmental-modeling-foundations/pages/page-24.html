5.3 Code pattern
The live script’s code pattern is conceptually:
t = 0:0.1:10;
k = 1;
Q0 = 1;
qi = Q0;
soln = ode45(@(t,y) ODEs(t,y), t, qi);
q_sol = deval(soln,t,1);
true_sol = Q0*exp(-k*t);
and the right-hand side is:
function derivVector = ODEs(t,y)
k = 1;
q = y(1);
dqdt = -k*q;
derivVector = [dqdt]';
end
This code teaches three habits:
– use an ODE function that takes t and y, even if t is not explicitly used;
– return a column vector, hence the transpose in [dqdt]';
– compare with an analytical solution when one exists.
