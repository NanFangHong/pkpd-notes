18A self-study route

From Matrix Tricks to MBAM


18.1 The order I would read
I would not start with MBAM directly. I would read in this order:
(i) 6.S087 probability and matrix tricks: random variables, covariance, gradi-
ents, Jacobians.
(ii) Matrix Cookbook: identities as a reference, not as a book to memorize.
(iii) 18.03: ODE modeling, units, linearization, qualitative behavior.
(iv) 18.031: LTI systems, convolution, transfer functions, poles.
(v) 18.335: conditioning, residual/error distinction, QR/SVD, iterative meth-
ods.
(vi) 6.441: divergence, local Fisher geometry, information projection.
(vii) 6.435: Bayesian workow, plate notation, approximate inference, and
model criticism.
(viii) MBAM papers: model manifold, geodesics, boundary limits.
(ix) QSP CRISP paper: how these ideas become a workow for pharmacological
models.
