3.3 State-space form
Linear compartmental models can be written as

q(t) = Aq(t) + Bu(t), y(t) = Cq(t).
Here q is the state vector, u is the input vector, and y is the output vector.
This notation is not merely cosmetic. It is the bridge to the later course topics:
– controllability asks whether inputs can move the state through relevant
directions;
– observability asks whether outputs contain enough information to recon-
struct states;
– identiability asks whether parameters can be inferred from outputs;
– stability asks whether trajectories decay, grow, or settle;
– sensitivity asks which parameter perturbations change outputs.
