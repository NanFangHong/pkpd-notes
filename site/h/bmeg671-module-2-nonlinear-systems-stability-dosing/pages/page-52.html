12.1 The symbolic workow
The stability live scripts show a useful MATLAB pattern:
(i) declare symbolic variables with syms;
(ii) write the right-hand side f(x, θ);
(iii) solve f (x, θ) = 0;
(iv) compute jacobian;
(v) substitute a stationary point and parameter values;
(vi) compute eigenvalues with eig;
(vii) scan parameters and record whether max Re λ < 0.
Mathematically this is simply the local stability theorem. Computationally it
becomes a map from parameter space to qualitative behavior.
The core code skeleton is short enough to keep in mind:
syms f r a b c d
df = a*r*f - b*f;
dr = c*r - d*r*f;
solution = solve(df==0, dr==0, [f;r]);
A = jacobian([df;dr],[f;r]);
ds = 0:0.01:1;
for i = 1:length(ds)
d = ds(i);
eig_value = double(subs(eig(A)));
stables(i) = max(real(eig_value)) < 0;
end
The line max(real(eig_value)) < 0 is the computational form of local asymp-
totic stability. The scan does not merely make a pretty gure; it asks whether
a qualitative conclusion survives parameter uncertainty.
