This is not a numerical trick. It is the mathematical statement that ODE

The vaccine live script makes the carry-over explicit:
theta = [da,lambda,db,e];
qi = [0,1];
[t1,x1] = ode15s(@(t,y) ODEs(t,y,theta,0), t_before, qi);
qi = x1(end,:);
[t2,x2] = ode15s(@(t,y) ODEs(t,y,theta,1), t_dose, qi);
qi = x2(end,:);
[t3,x3] = ode15s(@(t,y) ODEs(t,y,theta,0), t_after, qi);
x = [x1;x2;x3];
and the right-hand side is:
function derivVector = ODEs(t,y,theta,u)
da = theta(1); lambda = theta(2);
db = theta(3); e = theta(4);
a = y(1); b = y(2);
dadt = -da*a + u;
dbdt = lambda - db*b + e*a*b;
derivVector = [dadt,dbdt]';
end
This is the cleanest example in the module of separating mechanism from regi-
men. The same ODEs function is used before, during, and after dosing; only u
and the initial condition change.
