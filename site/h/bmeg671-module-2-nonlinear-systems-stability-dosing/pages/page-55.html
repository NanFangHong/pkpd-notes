13.1 The standard model
The Sep 24 lecture uses a standard virus-host model:
˙x = λ − dx − βvx,
˙y = βvx − ay,
˙v = ky − wv.
Here x is uninfected target cells, y infected cells, and v free virus. The parame-
ters have natural interpretations:
λ: target-cell production d: target-cell loss β: infection rate
a: infected-cell loss k: virion production w: virion clearance
The MATLAB script uses symbolic algebra to solve stationary points, compute
the Jacobian, and evaluate eigenvalues under numerical parameter values [
5].
The uploaded script encodes the workow transparently:
syms x y v lambda d beta a k w
dx = lambda - d*x - beta*v*x;
dy = beta*v*x - a*y;
dv = k*y - w*v;
solution = solve(dx==0,dy==0,dv==0,[x;y;v]);
A = jacobian([dx;dy;dv],[x;y;v]);
eig(A_at_equilibrium)
The scientic discipline is in the order. First write the mechanism, then solve
for possible operating points, then linearize, then interpret eigenvalues. If we
jump directly to simulation, we may miss a threshold or unstable equilibrium
that explains why the simulation behaves the way it does.
