3.3 Linearization is not the model
The lecture’s warning is subtle: linearization is local. It says what the nonlinear
system does near the selected point. It does not guarantee global behavior.
A tumor-growth model can be locally stable near carrying capacity and still
biologically impossible if it permits negative tumor mass elsewhere. A viral
model can have a locally stable infection equilibrium and still fail if its parameter
range violates known biology. Local stability is a useful lens, not a license to
stop thinking.
