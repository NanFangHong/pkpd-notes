It is a state at which the model’s instantaneous derivative vanishes. It is not
necessarily healthy, desired, or realistic. A disease-free equilibrium, endemic
equilibrium, high-tumor equilibrium, and toxic-drug steady state can all be
stationary points.
The workow is:
(i) solve f(x) = 0;
(ii) discard mathematically valid but biologically impossible states, such as
negative concentrations;
(iii) compute the Jacobian at each remaining point;
(iv) analyze eigenvalues or use a Lyapunov argument;
(v) interpret the result in the context of model purpose and parameter ranges.
