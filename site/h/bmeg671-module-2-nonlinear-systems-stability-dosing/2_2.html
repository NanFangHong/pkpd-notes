2.2 The SIR script as a bridge
The plain MATLAB SIR script is the bridge between hand-written equations
and the SimBiology project:
beta = 0.9;
a = 0.1;
S = y(1);
I = y(2);
R = y(3);
dSdt = -beta*S*I;
dIdt = beta*S*I - a*I;
dRdt = a*I;
derivVector = [dSdt,dIdt,dRdt]';
This code is valuable because it makes the reaction grammar inspectable. The
infection ux is a loss from S and a gain to I. The recovery ux is a loss from
I and a gain to R. If the same model is rebuilt in SimBiology, those gains and
losses should reappear as reactions and rate laws, not as a visually pleasing but
dierent mechanism.
